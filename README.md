# DA-Lab Website

Data Analytics Laboratory — Kwangwoon University

## Files
- `index.html` — Home
- `members.html` — Lab Members
- `research.html` — Research Areas
- `publications.html` — Publications
- `teaching.html` — Teaching
- `contact.html` — Contact
- `style.css` — Global styles
- `shared.js` — Shared nav & footer

## Deploy to GitHub Pages

1. Go to github.com → New repository → name it `dalab-site` (or `[username].github.io`)
2. Upload all files in this folder
3. Go to Settings → Pages → Source: `main` branch → `/root`
4. Your site will be live at `https://[username].github.io/dalab-site/`

## Customize

- Edit member names/info in `members.html`
- Add real publications in `publications.html`
- Update course info in `teaching.html`
- Replace placeholder names with real student names
